# PythonOPPMiniProject

SpringBoard mini python project

Banking System: Python OOP Mini Project

This project is a simplified banking system built using Python object oriented programming principles

The application runs from the command line and allows users to manage customers, accounts, employees, loans, and credit cards

The system supports core banking operations such as creating customers, opening checking or savings accounts, making deposits and withdrawals, viewing balances, and managing financial services

All data is stored using JSON files so information is retained between program runs

The project demonstrates key software engineering practices including structured OOP design, exception handling, logging to both console and file, and modular project organization

A UML class diagram is included to document system architecture and relationships between components, its built using mermaid

To run the program execute python main.py from the project root directory
